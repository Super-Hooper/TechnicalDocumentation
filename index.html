Types of 3D Printers

There are two main types of desktop 3d printers:

  • FDM (Fused Deposition Modeling)
  • SLA (Stereolithography)

FDM printers are typically cheaper and produce lower quality prints compared SLA. You will likely observe the individual layers in the FDM print versus a nearly smooth surface in an SLA print. FDM printers deposit plastic at a specified layer height. SLA printers instead use resin and special UV masks to cure the resin.

Slicing

Slicing is an intermediate step between a 3D file and the physical machine. Slicing is performed by software programs. 3D models are loaded into the software, and the slicer slices the model into the required layers. Since each printer has unique settings, slicing software has many configuration options to ensure optimum settings for the specific printer.

Some common slicer settings that the user can configure are:

  • Layer Height (sometimes referred to as quality or resolution
  • Enable supports
  • Enable adhesive raft or brim

G-Code

G-code is the code that the slicer produces that the 3D printer can understand. The code commands the printer temperature settings, extrusion, and motor position/speed. One thing to note is that slicers typically automatically produce a starting code with:

G28 ; Home

This command homes printer before the print begins. The issue is that if you have a printer that has auto bed leveling capability, the G28 command will disable the use of the bed mesh.

To fix this, you should configure your slicer settings to add the following g-code command after the home command:

M420 S1; Enable bed mesh

Here are some other sample g-code commands you might come across in the g-code file:

G0/G1 ;Rapid/Controlled Motion
G2 ;Clockwise Motion
G20/G21 ;Inches or Millimeters

Alignment

Alignment is critical in producing a quality print. If the alignment is not perfect, the first layer will not print evenly which can mess up the whole print. If your printer has auto bed leveling capability, you should first manually level the bed as close as possible. This will allow the auto bed leveling to be a little more reliable.

If you are having a difficult time with alignment of the nozzle to the bed, you may need to check the alignment of the 3D printer frame, specifically the X-Axis bar (which the extruder rides on).

Layer Height

Layer height is the height the printer moves up for each layer. Lower layer heights produce finer quality prints, with the trade-off of longer print time.

It is possible to adjust the height of the first layer in the slicer settings.
